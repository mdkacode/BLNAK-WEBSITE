Your phone is not a tool anymore. It is a slot machine dressed as a tool. Every time you unlock it,
dozens of apps compete to be the one thing you look at next — and every one of them is funded to
win that competition.

BLNAK is the smallest possible rebellion against that arrangement.

## The rules we set ourselves

1. **No network permission.** If the app cannot connect to a server, it cannot send anything
   anywhere. This is not a policy; it is a property of the binary.
2. **No growth features.** Nothing in BLNAK is designed to make you use BLNAK more. A launcher you
   open rarely is a launcher working as intended.
3. **No premium tier.** Locking features behind a paywall creates incentives we do not want.
4. **Open source forever.** The code is public so you can verify the first three.

## What a minimalist home screen feels like

When you open BLNAK for the first time, the home screen is blank. There is a search bar at the
bottom and nothing else. No wallpaper, no widgets, no clock. The silence is deliberate.

You type two letters, an app launches, you close it, you put the phone down. That is the entire
interaction loop. The launcher disappears between uses — which is exactly what a launcher should do.
